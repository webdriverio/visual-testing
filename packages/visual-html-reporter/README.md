# WebdriverIO Visual HTML Reporter

> A WebdriverIO Reporter for creating HTML-reports for the Visual Service.

## Installation

The easiest way is to keep `@wdio/visual-html-reporter` as a dev-dependency in your `package.json`, via:

```sh
npm install @wdio/visual-html-reporter --save-dev
```

Instructions on how to get started can be found in the [visual testing](https://webdriver.io/docs/visual-testing) docs on the WebdriverIO project page.
